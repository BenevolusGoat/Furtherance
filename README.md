# Furtherance
The Binding of Isaac: Furtherance is a modded expansion that adds a flavorful amount of new content!


## Features
- 14 Passive Items
- 18 Active Items
- 4 Trinkets
- 2 Characters
- 7 Blessings
- 1 Curse
- Mod Config Menu Support
- EID Suppport
- Encyclopedia Support
- Pog Support
- And more to come!

### Supported Languages (With EID)
- **English**
- **Spanish**
- **Russian**
